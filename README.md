# delloul.github.io

## 👋 Welcome to My Portfolio

This is the source code for my personal portfolio website, built with HTML5 and CSS3.

### 🚀 Features

- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Modern Aesthetics**: Clean and professional design with smooth animations
- **Optimized Performance**: Fast-loading static site hosted on GitHub Pages
- **Accessible**: Built with semantic HTML and WCAG accessibility standards in mind

### 📂 Structure

```
├── index.html       # Main portfolio page
├── styles.css       # Styling and responsive design
└── README.md        # This file
```

### 🎨 Sections Included

1. **Navigation Bar**: Easy navigation across all sections
2. **Hero Section**: Eye-catching introduction with call-to-action buttons
3. **About Me**: Personal introduction and professional philosophy
4. **Skills**: Organized into Creative & Design, Technical Development, and Soft Skills
5. **Experience**: Professional roles and key achievements
6. **Projects**: Featured projects with tech stacks and impact statements
7. **Education**: Certifications and professional development
8. **Contact**: Links to email, LinkedIn, and GitHub
9. **Footer**: Copyright information

### 🔧 Customization

To personalize this portfolio:

1. **Update Contact Information** in the Contact section:
   - Replace `your-email@example.com` with your actual email
   - Update LinkedIn and GitHub URLs

2. **Modify Project Details**:
   - Update project names and descriptions
   - Add project links
   - Update tech stacks

3. **Adjust Colors**:
   - Edit CSS variables in `styles.css` (Root Variables section)
   - Current color scheme uses blues and gradients

4. **Add New Sections**:
   - Follow the existing HTML structure
   - Use the same CSS classes for consistency

### 📱 Responsive Breakpoints

- **Desktop**: Full width layout
- **Tablet** (768px and below): Adjusted spacing and font sizes
- **Mobile** (480px and below): Single column layouts, optimized touch targets

### 🌐 Deployment

This site is automatically deployed to GitHub Pages at: `https://saifedelloul.github.io`

**Note**: Make sure to enable GitHub Pages in your repository settings:
1. Go to Settings → Pages
2. Select "Deploy from a branch"
3. Choose "main" branch and "/ (root)" directory
4. Save
